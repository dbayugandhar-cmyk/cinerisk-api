Brand scan
Scan for counterfeit sellers and fraud channels targeting a specific brand.
POST /v1/scan
Request body
brandstringrequiredBrand name to scan — e.g. "Nike", "Samsung", "Dove"
platformsarrayoptionalPlatforms to scan. Default: all. Options: telegram, indiamart, meesho, instagram, sharechat, youtube
languagesarrayoptionalLanguages to scan. Default: all. Options: en, hi, te, ta, kn, ml
min_scoreintegeroptionalMinimum risk score to include. Default: 35. Range: 0–100
{
  "brand": "Nike",
  "platforms": ["indiamart", "meesho", "telegram"],
  "min_score": 50
}
Response
{
  "brand": "Nike",
  "scan_id": "scan_20260510_084532",
  "scanned_at": "2026-05-10T08:45:32Z",
  "total_findings": 23,
  "findings": [
    {
      "platform": "indiamart",
      "seller_name": "BUDDY_HOUSE",

Seller risk score
Score a specific IndiaMART or Meesho seller URL for counterfeit risk.
POST /v1/risk/seller
Request body
urlstringrequiredIndiaMART or Meesho seller or listing URL
brandstringoptionalBrand being counterfeited — improves scoring accuracy
{
  "url": "https://www.indiamart.com/proddetail/nike-first-copy-shoes...",
  "brand": "Nike"
}

Batch seller scoring
Score up to 50 sellers in a single request.
POST /v1/risk/batch
Request body
sellersarrayrequiredArray of seller objects. Max 50 per request.
{
  "sellers": [
    { "url": "https://indiamart.com/...", "brand": "Nike" },
    { "url": "https://meesho.com/...", "brand": "boAt" }
  ]
}

Webhook registration
Register a URL to receive real-time alerts when new threats are detected.
POST /v1/webhooks/register
Request body
urlstringrequiredHTTPS URL to receive POST notifications
eventsarrayoptionalEvent types. Default: all. Options: counterfeit, piracy, betting, fraud, digital_arrest
brandsarrayoptionalFilter alerts to specific brands only
min_scoreintegeroptionalMinimum risk score to trigger webhook. Default: 50
{
  "url": "https://your-platform.com/cineos-webhook",
  "events": ["counterfeit", "piracy"],
  "brands": ["Nike", "Samsung"],
  "min_score": 60
}
Webhook payload (sent to your URL)
{
  "event": "counterfeit",
  "detected_at": "2026-05-10T14:22:10Z",
  "risk_score": 77,
  "brand": "Nike",
  "platform": "indiamart",
  "seller": "BUDDY_HOUSE, Sirsa HR",
  "url": "https://indiamart.com/...",
  "evidence_hash": "sha256:a3f4b2..."
}

Multi-platform scan
Scan all 13 platforms simultaneously for a brand or keyword.
POST /v1/multiscan
Request body
querystringrequiredBrand name, keyword or fraud pattern to scan across all platforms
deepbooleanoptionalIf true, scans dark web and private channel variants. Default: false
{
  "query": "Zerodha",
  "deep": true
}
Response
{
  "query": "Zerodha",
  "total_findings": 18,
  "by_platform": {
    "telegram": 14,
    "youtube": 2,
    "twitter": 2
  },
  "total_reach": 540284,
  "highest_risk": {
    "channel": "@Groww_Tips_Zerodha_Upstox",
    "subscribers": 540284,
    "risk_score": 88
  }
}
Live Shield — real-time monitoring
Activate continuous monitoring for a brand or event. Ideal for IPL match days, product launches.
POST /v1/live_shield
Request body
brandstringrequiredBrand or event name to monitor continuously
duration_hoursintegeroptionalHours to monitor. Default: 24. Max: 168 (1 week)
webhook_urlstringoptionalURL to receive live alerts during monitoring window
Create API key
Generate a new API key for a client or partner. Admin access required.
POST /admin/keys/create
Request body
client_namestringrequiredName of the client or partner organisation
tierstringrequiredAccess tier: brand, fraud_intel, enterprise
rate_limitintegeroptionalRequests per hour. Default by tier: brand=50, fraud_intel=100, enterprise=unlimited
