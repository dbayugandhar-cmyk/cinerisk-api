CINEOS Intelligence API

Real-time piracy detection for film, gaming, sports and India cinema. Detect piracy in under 500ms across 1000+ sources.

Base URL: https://cinerisk-api-production.up.railway.app

Introduction

The CINEOS API gives you programmatic access to real-time piracy intelligence. Scan any film, game, or sports event across 1000+ piracy sources with a single API call.

Every response includes the verdict, infringing URLs, platform names, and usage metadata. All endpoints return JSON.

Authentication

All API requests require an API key passed in the X-API-Key header.

curl -X POST https://cinerisk-api-production.up.railway.app/v1/scan \
  -H "X-API-Key: ck_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{"title":"Jet Lee","category":"film"}'

To get an API key, email dba.yugandhar@gmail.com or use the form below.

POST /v1/scan

Scan for piracy across 1000+ sources. Returns verdict, infringing URLs, and platform list.

POST /v1/scan

Request Body

{
  "title": "Jet Lee",        // required — film/game/event name
  "category": "film",    // film | india | gaming | sports
  "language": "telugu"   // optional — any | hindi | telugu | tamil
}

Response

{
  "success": true,
  "data": {
    "title": "Jet Lee",
    "verdict": "CONFIRMED",
    "hits_found": 3,
    "platforms": ["5movierulz.camera", "thepiratebay.org"],
    "hits": [{
      "platform": "5movierulz.camera",
      "url": "https://www.5movierulz.camera/jetlee-2026-telugu/...",
      "title": "JetLee (2026) Telugu Full Movie Watch Online"
    }]
  },
  "meta": {
    "response_time_ms": 323,
    "queries_used": 1,
    "queries_remaining": 4999,
    "tier": "pro"
  }
}

POST /v1/live_shield

Scan Telegram channels for illegal sports streams. Detects IPL, cricket, Premier League streams with language detection.

POST /v1/live_shield 
{
  "event_name": "IPL 2026"
}

{
  "data": {
    "streams_found": 7,
    "critical": 3,
    "streams": [{

      "language": "Tamil",

      "is_betting": false,
      "severity": "HIGH"

  }
}
